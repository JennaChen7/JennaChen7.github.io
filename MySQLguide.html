MySQL Installation Guide
I was trying to build a login interface for this website. So first it was like...

Ok.
First I need to create a database using MySQL. For Mac users with the M1 chip, MySQL Branch may not be downloaded successfully.
I used MySQL instead.
So you can download it from the offical site(I used a mirror so it works faster in my country[http://mirrors.sohu.com/mysql/].

Then, just continue with the installation...

Here on the configuration page select the second one and set your password.

After you finish installation nothing will show up on the desktop(Don't worry it's working).
Then, open terminal and type in the following two lines:

'cd /'
'sudo vim ./.bash_profile'

(Don't include the quotation mark)
It will than ask you to enter your password(the one you set during installation)
The password WILL NOT SHOW UP on the line. When you're done just hit enter.

Then it will bring you to a page looking like this:

Press i and the last line will say "INSERT" in bold.
Then, enter these two lines:

'export PATH=$PATH:/usr/local/mysql/bin
export PATH=$PATH:/usr/local/mysql/support-files'

Press esc and then type ': wq', which will show up on the last line.
Hit enter.

If everything went right you should be back on the page where you just typed 'sudo vim ./.bash_profile'.
Then, enter this line:

'source ~/.bash_profile'

Nothing will show up because this only refresh the environment. After this, enter this line:

'mysql -u root -p'

and enter your password again.
If nothing went wrong something like this should show up:


Finished!
